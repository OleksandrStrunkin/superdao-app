# Superdao Landing Page

A simple pet project landing page inspired by a Web3/DAO product style.
Built for practice in layouting, responsive design, and Tailwind workflow.

## Tech Stack

- HTML5
- Tailwind CSS
- Vanilla JavaScript

## Features

- Responsive layout (mobile/tablet/desktop)
- Reusable color utility classes
- Fluid typography with `clamp()`
- Mobile burger menu

## Getting Started

### 1) Install dependencies

```bash
npm install
```

### 2) Run Tailwind in watch mode

```bash
npm run tw:watch
```

### 3) Open the page

Open `index.html` in your browser (or use Live Server in VS Code/Cursor).

## Available Scripts

- `npm run tw:watch` - watches files and rebuilds `src/output.css`
- `npm run tw:build` - creates minified production CSS

## Project Structure

- `index.html` - main page markup
- `input.css` - Tailwind input + custom utilities
- `src/output.css` - generated Tailwind CSS
- `src/script.js` - simple UI interactions
- `src/img/` - images and icons
